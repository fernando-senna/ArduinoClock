# ArduinoClock
Clock using an Arduino UNO R3, LCD1602, &amp; RTC DS3231

<h2>Parts List</h2>
<ul>
  <li>Arduino UNO R3</li>
  <li>LCD1602 16x2</li>
  <li>RTC DS3132</li>
  <li>Push Buttons (5)</li>
  <li>On/Off Switch</li>
  <li>10k&Omega; Resistor (6)</li>
</ul>
